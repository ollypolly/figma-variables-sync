### 🛠️ Configuration & Code Changes

To ensure TypeScript 6 and Vite 7 compiled cleanly, the following configurations were modified:
- **`tsconfig.json`**:
  - Added `"ignoreDeprecations": "6.0"` to keep `"moduleResolution": "Node"` active for resolving older dependencies (e.g. `monorepo-networker`) that lack modern export map structures.
  - Updated `"typeRoots"` and `"types"` keys to correctly resolve both standard `@types/*` modules and `@figma/plugin-typings` under TypeScript 6.
- **`tsconfig.node.json`**:
  - Updated `"moduleResolution"` to `"Bundler"` to correctly resolve modern library types such as `@vitejs/plugin-react` within root configuration files.
- **`vite.config.ui.ts`**:
  - Removed the redundant `preprocessorOptions.scss` block (`api: "modern-compiler"`) since Vite 7 activates the modern Sass compiler API by default.
